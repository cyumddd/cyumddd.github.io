Hexo个人博客搭建教程

Hexo个人博客搭建教程

准备工具

nodejs

下载连接：https://nodejs.org/zh-cn/download

安装选项说明：

  • 默认勾选了核心组件：Node.js runtime（运行时）、corepack manager（包管理器）、npm（Node包管理工具）和PATH环境变量配置
  • 核心运行时(node.exe)需要81MB硬盘空间

推荐操作：

  • 新手用户保持默认选项直接点击”Next”即可
  • 高级用户可根据需求：
    • 取消勾选npm（若已单独安装或想使用yarn/pnpm）
    • 调整安装路径（通过”Browse”按钮）
    • 查看磁盘使用情况（”Disk Usage”按钮）

注意事项：

  • 建议保留”Add to PATH”选项以便全局使用Node命令
  • 安装完成后在命令行运行 node -v 和 npm -v 验证安装

自动安装必要工具。注意：此操作会同时安装 Chocolatey。安装完成后，脚本将在新窗口中弹出。

至此安装完成

git

下载链接：https://git-scm.com/downloads

https://registry.npmmirror.com/binary.html?path=git-for-windows/

进来之后选择自己合适的版本即可，这里提供了两种类型的git：

独立安装程序（Standalone Installer）：32位和64位Git for Windows的安装程序。独立安装程序是一种简便的方式，将Git for Windows直接安装到计算机上，使其成为系统的一部分。
便携版（Portable）：32位和64位Git for Windows的便携式版本，也被称为“存储在可移动设备上的版本”。便携版可以在没有安装过程的情况下直接运行，非常适合携带在便携式存储设备（如USB闪存驱动器）中使用，方便在不同计算机之间使用Git。

一般选择64位的安装包即可。
